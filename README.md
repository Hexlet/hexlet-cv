# hexlet-cv

### Участие

* Обсуждение в канале #ruby слака http://slack-ru.hexlet.io

### Required

* Ruby
* Node
* Yarn
* sqlite3
* [heroku cli](https://devcenter.heroku.com/articles/heroku-cli#download-and-install)

### Setup

```sh
$ make setup
$ make test # run tests
$ make start # run server
```

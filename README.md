# hexlet-cv

### Участие

* Обсуждение в канале #ruby слака http://slack-ru.hexlet.io

### Requirements

* Mac / Linux
* Docker
* Docker Compose

### Install

```sh
$ make compose-setup
```

### Run

```sh
$ make compose
```

Go to http://localhost:8080/
